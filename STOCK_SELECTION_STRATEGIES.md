# 🏛️ 基本面選股策略

基本面分析著重於公司的財務體質與獲利能力，適合**長期投資**與**價值投資**者。

---

## 1. 本益比 (P/E Ratio) 篩選

### 原理
本益比 = 股價 ÷ 每股盈餘 (EPS)
- 代表「投資幾年可以回本」
- 數值越低，可能越便宜（但也可能是衰退股）

### 篩選條件
| 條件 | 說明 |
|------|------|
| P/E < 15 | 低於大盤平均 |
| P/E > 0 | 排除虧損公司 |
| 近 4 季 EPS > 0 | 獲利穩定 |

### 注意事項
- 不同產業本益比差異大（科技股通常較高）
- 建議與「同業」比較，而非絕對數值

---

## 2. 股價淨值比 (P/B Ratio) 篩選

### 原理
股價淨值比 = 股價 ÷ 每股淨值
- P/B < 1 表示股價低於公司帳面價值
- 常用於「撿便宜」策略

### 篩選條件
| 條件 | 說明 |
|------|------|
| P/B < 1.5 | 相對低估 |
| 每股淨值 > 10 | 帳面資產有一定規模 |
| ROE > 8% | 避免選到「資產多但不賺錢」的公司 |

---

## 3. 股息殖利率篩選 (高股息策略)

### 原理
殖利率 = 現金股利 ÷ 股價 × 100%
- 高殖利率代表穩定配息能力
- 適合追求現金流的投資人

### 篩選條件
| 條件 | 說明 |
|------|------|
| 殖利率 > 5% | 高於定存利率 |
| 連續 5 年配息 | 配息穩定 |
| 配息率 < 80% | 保留盈餘再投資 |

### 資料來源
- 證交所個股日本益比、殖利率及股價淨值比
- https://www.twse.com.tw/zh/trading/statistics/index04.html

---

## 4. ROE (股東權益報酬率) 篩選

### 原理
ROE = 淨利 ÷ 股東權益 × 100%
- 衡量公司運用股東資金的效率
- 巴菲特最愛的指標之一

### 篩選條件
| 條件 | 說明 |
|------|------|
| ROE > 15% | 優質公司門檻 |
| 連續 3 年 ROE > 10% | 獲利能力穩定 |
| 負債比 < 50% | 財務結構健全 |

---

## 5. 營收成長篩選

### 原理
營收是獲利的源頭，持續成長的公司通常股價也會跟著成長。

### 篩選條件
| 條件 | 說明 |
|------|------|
| 月營收年增率 > 10% | 成長動能 |
| 連續 3 個月正成長 | 趨勢確認 |
| 累計營收年增率 > 5% | 全年表現 |

---

## 6. 毛利率與營益率篩選

### 原理
- **毛利率** = (營收 - 成本) ÷ 營收 → 產品競爭力
- **營益率** = 營業利益 ÷ 營收 → 本業獲利能力

### 篩選條件
| 條件 | 說明 |
|------|------|
| 毛利率 > 20% | 產品有競爭力 |
| 營益率 > 10% | 本業賺錢 |
| 近 4 季毛利率穩定 | 沒有大幅衰退 |

---

## 7. 自由現金流篩選

### 原理
自由現金流 = 營業現金流 - 資本支出
- 正值表示公司賺的錢足以支應投資
- 長期穩定正值是優質公司的特徵

### 篩選條件
| 條件 | 說明 |
|------|------|
| 自由現金流 > 0 | 連續 3 年正值 |
| 營業現金流 > 淨利 | 獲利品質好 |

---

## 8. 低負債高現金篩選

### 原理
安全性第一，選擇財務體質穩健的公司。

### 篩選條件
| 條件 | 說明 |
|------|------|
| 負債比 < 40% | 財務結構穩健 |
| 流動比 > 150% | 短期償債能力強 |
| 現金佔總資產 > 10% | 抗風險能力 |
